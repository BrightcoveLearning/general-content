Video Cloud

Video Cloud Courses
Media Management with Brightcove Video Cloud https://university.brightcove.com/catalog/courses/681885
In this course, you will learn how to manage your content to create the best possible experience for your viewers using the Upload and Media Modules in Video Cloud.
Creating and Scheduling Custom Analytics Reports https://university.brightcove.com/catalog/courses/681885
Custom analytics reports allow you to define exactly what you see in Analytics reports and to run them either once or on a daily, weekly, or monthly schedule. Reports are available in CSV or XLSX format, and you can get notifications of new reports emailed to yourself and others. This course shows you how to create and manage custom reports.
Administering Video Cloud https://university.brightcove.com/catalog/courses/512605
Administering Video Cloud provides an introduction to many of the administrative functions you can perform in the Admin section of Studio. It will show you how to add additional users, create custom video metadata fields, share videos between multiple Video Cloud account, manage ingest profiles that control how your videos are processed when you upload them, and how to create client credentials that can be used to authenticate API requests if you have developers who will be using Brightcove's APIs.
Introduction to Video Cloud Analytics https://university.brightcove.com/catalog/courses/511393
Viewing the Introduction to Video Cloud Analytics training course is a great way to become familiar with using Video Cloud Analytics. This course was designed for publishers who are just getting started with Video Cloud.
Introduction to Video Cloud https://university.brightcove.com/catalog/courses/499241
Introduction to Video Cloud is an absolute must for those who upload, configure, and maintain online media. Learn valuable best practices that will help you save time and resources. This course is a packed session designed to meet the needs of your content producers, project managers, and editors.
Introduction to Brightcove Player https://university.brightcove.com/catalog/courses/841750
Welcome to Introduction to Brightcove Player.

For this course, we recommend that you take the Media Management with Brightcove Video Cloud course first.

In this course you will learn:

  1. How to create a new player in Video Cloud
  2. How to configure your player:
    • Edit player properties and customize the player appearance
    • Add plugins
    • Learn more about the player configuration page
    3. How to preview and publish content
    • Preview and publish changes in the player
    • Publish a video in the media module with the new player
